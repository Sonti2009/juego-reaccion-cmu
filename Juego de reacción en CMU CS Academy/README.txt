# 🎮 Reaction Game

Juego hecho en Python con CMU CS Academy.

## 🚀 Características
-Sistema de vidas
-Tiempo limite
-Dificultad progresiva
-High score
-Multiples objetivos

## 🎯 Cómo jugar
-Presiona espacio para iniciar
-Haz click al circulo correcto
-Evita perder vidas

## 🧠 Lo que aprendí
-Manejo de eventos
-Logica de estados
-Uso de grupos y condicinales
